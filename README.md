## Visual1 Starter Kit

1. statamic new {{ project_name }} visual1/starter-kit
2. cd {{ project_name }}
3. cp .env.example .env
4. composer require statamic/eloquent-driver
5. Create mysql database
6. artisan migrate --seed
7. npm install
8. npm run dev
